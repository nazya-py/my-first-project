## Deskripsi

Program sederhana untuk mengecek apakah seorang user adalah "sultan" berdasarkan saldo yang mereka miliki.

## Fitur

- Input nama user
- Input jumlah saldo
- Pengecekan kondisi (if/else)
- Output pesan yang dinamis

## Kode Program

```python
print('Nazya jadi sukses')
nama = input('Nama kamu siapa: ')
saldo = input('Saldo kamu berapa: ')

if int(saldo) >= 500000:
    print(f'{nama} udah jadi sultan asli tajir melintir')
else:
    print(f'{nama} masih calon sultan, gas nabung broo')
```

## Cara Menggunakan

1. Jalankan program
2. Masukkan nama kamu
3. Masukkan jumlah saldo kamu
4. Program akan menampilkan status "sultan" kamu 😄
